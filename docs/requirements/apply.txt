KEY UX RULES
No native ClickUp public form.
Dropdowns must open cleanly and never cover questions.
Every dropdown that may require flexibility should include Other.
Every Other option should reveal a text field.
Progress bar must show where the user is.
Mobile must be clean and easy to complete.
Buttons must remain clear:
Back
Continue
Save & Continue Later
Submit Diagnostic
Keep the experience tight.
The user should never feel like they are filling out a long government form.
________________


DESIGN STANDARD
This page should feel:
* private

* serious

* premium

* confidential

* structured

* intelligent

* application based

* mobile first

Visual tone:
   * black

   * champagne gold

   * dark glass

   * subtle Axis light

   * thin dividers

   * secure interface

   * premium form fields

   * controlled spacing

Avoid:
      * native ClickUp forms

      * long scrolling forms

      * generic survey design

      * bright form colors

      * crowded fields

      * weak confirmation screens

      * casual wording

Final feeling:
The user is not filling out a form. They are entering the Axis evaluation system.
